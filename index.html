    Kumiko 学习笔记小站

    如题，Kumiko 发布了一些自认为做的比较认真的科目的笔记（的阉割版本，去掉了大部分的练习题和考试题）

    Kumiko 在整理这些笔记的时候确实比较摆烂，有些地方直接截图教材和 PPT，有些地方省略了很多，有些地方却花很大篇幅讲了一些没有用的东西，还有一些东西是当时学的时候所想，现在一看却是错的……

    对于学弟学妹来说

    • 不要尽信这份笔记，还是以教材为准；
    • 如果想自己做 LaTeX 笔记或者 Markdown 笔记，可以参考源码，节省一些打字的时间；
    • 欢迎在评论区分享学习资料，也可以提出修改建议，我会尽力修改。

    关于 PR

    由于笔记是我本地用 Typora 编译的，所以只能把源文件发给我，我再进行编译，发到 github 上，不过这样你就不会出现在 contributor 里面了。

    对于版权所有者

    写这份笔记的时候确实参考了很多教科书、参考书、老师的 PPT、B 站视频等，如果您认为侵权，请联系我删除。
